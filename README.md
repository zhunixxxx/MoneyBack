# MoneyBack

本地出差报销整理工具。根据报销标准模板创建报销文件夹，通过浏览器拖拽自动复制文件到对应目录。

## 功能

- **报销模板**：配置住宿标准（一线城市 / 其他地区）和餐饮标准（早/午/晚）
- **创建报销**：在指定根目录下自动生成结构化文件夹
- **拖拽上传**：将发票、行程单、水单等文件拖入浏览器，自动复制并按规则重命名
- **出差类型**：交通费（高铁、打车）、住宿费、餐饮

> 公出（市内）场景暂未实现。

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式（前端 + 后端）
npm run dev
```

- 前端：http://localhost:5173
- 后端 API：http://localhost:3456

## 使用流程

1. **设置** → 配置报销根目录（默认 `~/MoneyBack报销`）
2. **报销模板** → 确认或修改住宿/餐饮标准
3. **报销列表** → 新建报销，选择模板
4. 进入报销详情 → 将文件拖拽到对应区域

## 技术栈

- 前端：React + TypeScript + Vite + Tailwind CSS
- 后端：Express + TypeScript
- 数据：本地 JSON 文件（`server/data/app.json`）
